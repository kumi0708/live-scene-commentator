# Live Scene Commentator

iOSのカメラを使って、1秒ごとにシーンを説明する
**AI Edge Gallery Skill（擬似リアルタイム）**

## Features

* カメラ起動（iOS）
* 1秒ごとの静止画キャプチャ
* シーン説明の自動更新
* 擬似リアルタイム体験

## How it works

Camera → Capture → Vision Model → Text Description

## Usage

AI Edge Gallery の Skills から URL を指定して読み込み

## Notes

* iOS版はURLロードが不安定な場合あり
* ローカルファイル読み込みも推奨
